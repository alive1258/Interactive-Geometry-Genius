1.What is DOM and what is the purpose of DOM?

Ans: DOM means Document Object Model. It can be represented as a tree of elements created by the browser.It's change the HTML document. With DOM manipulation, it can create, modify, style, or delete elements without a refresh. It also promotes user interactivity with browsers.

2.How will you select HTML elements using DOM? Name the DOM methods?

Ans: DOM Selectors, as the name suggests is used to select HTML elements within a document using JavaScript. There are 5 ways in which you can select elements in a DOM using selectors.

(i).getElementsByTagName()

(ii).getElementsByClassName()

(iii).getElementById()

(iv).querySelector()

(v).querySelectorAll()

3.What is event bubble?

Ans: Event Bubbling is a concept in the DOM. It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy.its parent and ancestor elements in the DOM tree until it gets to the root element.

4.What is a callback function and why will you use it?

Ans: A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

Need callback function : Callbacks are important for anyone using server-based technology to provide real-time reporting for business intelligence purposes. From a purely mechanical perspective, callbacks are the method by which data is pulled from a server to another.
